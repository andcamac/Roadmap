
## Estructura Legal: El SPV

**SPV = Vehículo de Propósito Especial**

```
                    ┌─────────────┐
                    │  WOLI CBD   │
                    │    S.A.     │
                    └──────┬──────┘
                           │ manufactura
                           ▼
                    ┌─────────────┐
                    │     SPV     │ ← entidad legal separada
                    │ (inventario │
                    │ como garantía)│
                    └──────┬──────┘
                           │ tokens
                           ▼
                    ┌─────────────┐
                    │  Inversores │
                    │   RWA       │
                    └─────────────┘
```

Los holders de tokens son dueños de acciones del SPV, **no de Woli CBD S.A.** Esta separación legal es crítica.
